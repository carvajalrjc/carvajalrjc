## Featured Projects

### Secure Ticket - Web2.5 Digital Ticketing Platform on Stellar

Academic thesis project built as a full-stack and blockchain prototype for digital ticketing. It combines a traditional web experience with signed QR validation, controlled resale, smart contracts, blockchain event indexing, an admin panel, and automated tests.

**Tech stack:** React, Vite, TypeScript, Tailwind CSS, Node.js, Express, Prisma, PostgreSQL/Supabase, Rust, Stellar/Soroban, Stellar SDK, Freighter Wallet, JWT, Playwright, Vitest, Supertest, k6, Vercel, Railway.

**Highlights:**

- Implemented 53 backend routes/handlers and 25 database models.
- Built 3 Soroban smart contracts for the blockchain layer.
- Added backend, frontend, contract, E2E, and load testing coverage.
- Designed flows for signed QR tickets, scanner validation, P2P resale, NFT collectible, PQR module, and admin management.

Repository: [github.com/Tesis-Stellar/Secure-Ticket](https://github.com/Tesis-Stellar/Secure-Ticket.git)

### Academic Project Management Platform

Microservices-based academic project management platform developed with NestJS, Next.js, Docker Swarm, and GitHub Actions. The project focused on service decoupling, independent deployments, CI/CD pipelines, container orchestration, and teamwork in a 12-person academic environment.

### Arrienda tu finca

Client-server web application built with Spring Boot and Angular. The project applied REST APIs, Scrum, and DevOps practices during the development cycle.

### Loan Approval Classification

Machine learning project using Python and scikit-learn to train and evaluate SVM and multilayer perceptron models over a dataset of approximately 45,000 records, applying EDA, preprocessing, cross-validation, Accuracy, F1-score, and ROC-AUC.
